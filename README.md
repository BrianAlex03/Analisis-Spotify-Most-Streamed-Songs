Dataset link in kaggle https://www.kaggle.com/datasets/abdulszz/spotify-most-streamed-songs
Análisis exploratorio y predictivo sobre  dataset de las canciones más reproducidas en Spotify. Se lleva a cabo un proceso de limpieza y preparación de datos, incluyendo el tratamiento de valores nulos y el análisis de correlación entre variables musicales y métricas de popularidad.
Herramientas utilizadas: Python; Librerias como Pandas, NumPy, Matplotlib, Seaborn y Scikit-learn.
